## How It Works

### 1. Voice Guide Integration
The script loads your `VOICE_GUIDE.md` and passes it to Sonnet as a system instruction. This ensures all customizations match your principal-level tone exactly:
- Strategic, outcome-driven language
- Systems thinking emphasis
- Business awareness + calm confidence
- No hype or junior-level passion statements

### 2. Job Posting Analysis
Sonnet receives the job posting content and extracts:
- Key responsibilities
- Required skills
- Company mission/values
- Role-specific keywords

### 3. Cover Letter Customization
Sonnet generates a principal-level cover letter that:
1. Opens with strategic introduction
2. Includes 1-2 specific initiatives with quantified impact
3. Demonstrates leadership and mentoring
4. Aligns your systems thinking with the company's challenges
5. Uses your authentic voice from the guide
6. 4-5 tight paragraphs (not overly long)

### 4. Resume Customization
Sonnet reorders your base resume to:
- Lead with most relevant experience
- Front-load achievements matching the job
- Integrate keywords naturally
- Highlight leadership and systems thinking
- Preserve quantified impact
- Maintain ATS optimization

### 5. Output Generation
Both customized documents are saved as text files in:
```
customized_applications/[Company]/
  ├── Cover_Letter_[Company].txt
  └── Resume_[Company].txt
```

### 6. Email Delivery (Future)
When integrated with OpenClaw's message tool, the script will:
- Email both customized documents to ryanwinzenburg@gmail.com
- Include a summary of what was customized
- Provide next steps (review → edit → submit)

## Troubleshooting

### "ANTHROPIC_API_KEY not set"
```bash
export ANTHROPIC_API_KEY="sk-ant-your-key-here"
```

### "anthropic module not found"
```bash
pip3 install anthropic
```

### "API Error: Invalid API key"
Check that your key is correct and has valid credits/quota.

### "Job posting fetch failed"
Currently the script uses a placeholder. When integrated with OpenClaw's web_fetch, it will fetch real content.
